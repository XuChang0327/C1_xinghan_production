# C1 星瀚作品代码（产研 API）

本仓库保存 **C1 星瀚** 系列作品示例代码，基于 **产研 R2 API**（如 `r2_encoder_motor`、`r2_sound_sensor` 等模块），用于在配套主控与固件环境下连接传感器、电机与灯环等外设。

## 目录结构

|------|------|
| `J年级代码/` | J 年级作品脚本 |
| `S年级代码/` | S 年级作品脚本 |
| `test.py` | 本地调试/传感器与电机简单联调示例 |

## 运行环境

- **硬件**：需使用支持产研 R2 API 的控制器及对应固件；各脚本开头的 `print` 中会说明 **EncoderMotor / SoundSensor / RGBRing / DistanceSensor / Remoter** 等接口与端口（如 A、B）连接方式。
- **软件**：在设备提供的 **MicroPython / 配套运行环境** 中执行；`r2`、`r2_*` 模块由固件或设备侧提供，**无法在普通电脑仅装 pip 依赖后直接运行**整段逻辑。

使用前请按课程或硬件手册完成接线，再运行与作品对应的 `.py` 文件。
